How to Update the Database
  • Step 1 — Compress the new .dat files
    Upload testmat.dat and testply.dat to Claude and ask it to compress them. Download the resulting testmat.dat.gz and testply.dat.gz.
  • Step 2 — Upload to GitHub
    Go to the repo on GitHub. Click Add file → Upload files. Upload both .gz files and commit the changes.
  • Step 3 — Done!
    The GitHub Action runs extract_data.py automatically and commits the updated data.js back to the repo. Monitor progress under the Actions tab.
⚡

Fully automated: Once the .gz files are pushed, the workflow handles everything — no manual script running required.
